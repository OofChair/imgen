Endpoints

REQUEST https://imgen.herokuapp.com/{endpoint-name}

Supported Query Parameters:

text: String. Text to show on the generated image.

usernames: String. Username(s) to show on the generated image.

avatars: String. Avatar(s) to show on the generated image.

Headers

HEADERS All requests must be made with necessary headers

avatars: Dict. {'avatars': 'link1,link2'}

usernames: Dict. {'usernames': 'username1,username2'}

text: Dict. {'avatars': "hi,how are"}
